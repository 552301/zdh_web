主要功能
v1.0.02020.04.01
zdh 主要的作用 是从hdfs,hive,jdbc,http-json接口 等数据源拉取数据,并转存到hdfs,hive,jdbc等其他数据源 支持集群式部署
  1. 支持sql标准函数
  2. 支持界面选择配置
  3. 支持快速复制已有任务
  4. 支持外部调度工具(需要修改,新增特定接口)
  5. 弹性扩展(可单机,可集群)
  6. 支持客户级权限
  7. 简单易用支持二次开发
  8. 自带简单调度工具,可配置定时任务,时间序列任务,设定次数
  9. 支持SHELL 命令检查文件 if [ ! -f "/dir1/file1" ];then echo "文件不存在"; else echo "true"; fi
支持的数据源

  4. hbase
  5. mongodb
  6. es
  7. kafka
